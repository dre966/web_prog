# Gauntlet Database
----
This is a project im using to practice my web programming skills(MYSQL, PHP, JS).
This project is basically a journal for my gauntlet(a mode in Asphalt Legends Unite where racres try to beat each other's times) times so i know which tracks im good at... I use a MYSQL Database to keep track of users and cars they've selected as well as thier times so i can populate the UI with Javascript whilst PHP handles the server and database communication.
----
There are three pages in this site:
 - login.php(to create a user or login as one)
 - index.php(to select cars or modify car selection if already selected)
 - table.php(to display times set by user in either table form or block form)
